欢迎来到我的个人博客！这是一个使用 Next.js 构建的现代化博客网站。

## 为什么要创建这个博客？

作为一名前端开发者，我一直在寻找一个既能满足技术探索需求，又能提供良好用户体验的博客平台。经过反复思考，我决定自己动手搭建一个。

## 技术栈选择

我选择了以下技术栈来构建这个博客：

- **Next.js 14** - React 框架，提供服务端渲染和静态生成
- **TypeScript** - 类型安全的 JavaScript
- **Markdown** - 文章编写和存储格式
- **CSS Modules** - 样式管理
- **Vercel** - 部署平台

## 博客特性

这个博客具备以下特性：

1. **静态生成** - 快速的加载速度和优秀的 SEO
2. **响应式设计** - 在各种设备上都有良好的显示效果
3. **Markdown 支持** - 简单易用的文章编写方式
4. **代码高亮** - 技术文章中的代码块有语法高亮
5. **搜索友好** - 搜索引擎优化

## 接下来的计划

我计划在这个博客中分享：

- 前端开发技术心得
- 开源项目介绍
- 学习笔记和总结

## 如何开始写文章？

要添加新文章，只需要在 `posts` 目录中创建一个新的 Markdown 文件，并在文件头部添加必要的元数据：

```markdown
---
title: "您的文章标题"
date: "2024-12-27"
description: "文章描述"
---

您的文章内容...
```

文章内容支持标准的 Markdown 语法，包括标题、段落、列表、代码块等。
